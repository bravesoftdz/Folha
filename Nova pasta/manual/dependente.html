Definição

É uma entidade subordinada ao funcionário. Cada funcionário poderá ter nenhum, um ou vários dependentes. Cada dependente pertencerá a um único funcionário e poderá assumir vários tipos simultaneamente.

Objetivo

O recurso de dependentes visa fornecer suporte ao cálculo de eventos que necessitam da quantidade de dependentes que satisfaçam certos requisitos, como por exemplo um limite de idade.

Cadastro

O cadastro de dependentes estar disponível através da opção <Dependentes> do menu <Cadastro>. Todos os campos do dependente são obrigatórios. O dependente possui os sequintes campos:

  • Empresa

    Código da empresa a qual o funcionário está subordinado. O usuário não informa esse número, sistema assume o código da empresa atual.

  • Funcionario

    Código do funcionário que o dependente está subordinado. O usuário normalmente escolhe o funcionário.

  • Nome

    Nome completo do dependente conforme registrado na certidão de nascimento. Este campo identifica o dependente para o funcionário por isso não haverá mais de um dependente com o mesmo nome para o funcionário.

  • Nascimento

    Data do nascimento conforme registrado na certidão de nascimento. A data de nascimento é importante porque alguns tipos podem possuir limites de idade para considerar o dependente numa contagem.

  • Sexo

    O sexo do dependentes pode ser representado pelas letras "F" e "M" sendo respectivamente feminimo e masculino.

  • Tipos

    Texto alfanumérico contendo a relação de tipos que o dependente pertece. Cada letra representa um tipo, não pode ser usada as letras D, F, M e I. Se o usuário desejar pode inserir no texto outros caracteres que não sejam letras.

  • Inválido

    Indica se o dependente possui algum tipo de deficiência que o impossibilite de trabalhar. O dependente que possuir este campo marcado será considerado na contagem do tipo independentemente de sua idade.

Tipo de Dependentes

O tipo de dependente é uma classificação que tem por objetivo estabelecer limites de idade para a contagem de dependentes. O tipo é representado por uma letra, exceto "D", "F", "M" e "I". Cada dependente pode ser classificado em vários tipos simultanêamente.

Por padrão, o FolhaLivre já traz cadastrado 2 (dois) tipos de dependentes, que são:

Tipo Descrição Limite-Idade
S Salário Família 14 anos
R Imposto de Renda 21 anos

Para que um dependente seja classificado simultaneamente nestes dois tipoS, informe no campo TIPOS o texto "R-S".

O cadastro de tipos de dependentes estar disponível através da opção <Tipos de Dependente> do menu <Cadastro>.

Fórmula

A contagem dos dependentes do funcionário se realizará por intermédio de fórmula através da chamada de funções e constantes apropriadas.

As funções de dependentes iniciam-se pela letra "D", que são as sequintes:

  • D[T] ou D([T])

    Executa a contagem dos dependentes do tipo T, considerando o limite de idade para o tipo informado. Os dependentes cujo campo [] Inválido está marcado são incluídos na contagem, desconsiderando a idade. Veja os exemplos nas fórmulas para imposto de renda e salário-família respectivamente:

    d := DR*TGI(2,1)  // Dedução por dependentes - Tipo "R"

    VR := DS  // DS - Quantidade de dependentes do tipo S

  • D[T][M,F][I] ou D([T],[M,F],[I])

    Realiza a contagem dos dependentes considerando o limite de idade quando o tipo for especificado. Se o modificador opcional [M,F] for usado será considerado apenas os dependentes de sexo correspondente. Se o modificador opcional [I] for utilizado será considerado apenas os dependentes 'inválidos'.

  • DD ou DD()

    Executa a contagem geral dos dependentes desconsiderando quaisquer informações.

  • DD[T][M,F][I] ou DD([T],[M,F],[I])

    Realiza a contagem dos dependentes desconsiderando as idades e os limites do tipo. Se o modificador [T] for informado será considerado apenas os dependentes dos tipos [T]. Se o modificador opcional [M,F] for usado será considerado apenas os dependentes do sexo correspondente. Se o modificador opcional [I] for utilizado será considerado apenas os dependentes "inválidos".

Tabelas

São duas as tabelas que dão suporte ao recurso de dependentes.

  • A tabela F_DEPENDENTE_TIPO armazena os tipos de dependentes, onde cada linha representa um tipo. Possue a sequinte estrutura:

    Name Field

    Type

    # IDTipo

    Char(1)

    Descricao

    Varchar(30)

    Limite_ano

    SmallInt

  • A tabela F_DEPENDENTE armazena todos os dependentes do sistema, onde cada linha representa um dependente. Possue a sequinte estrutura:

    Name Field

    Type

    # IDEmpresa

    Integer

    # IDFuncionario

    Integer

    # Nome

    Varchar(50)

    Nascimento

    Date

    Tipo

    Char(10)

    Invalido_x

    SmallInt

    Sexo

    Char(1)
