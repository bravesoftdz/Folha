Os arquivos produzidos pelo Projeto FolhaLivre estão disponíveis em dois locais oficiais, a saber:

  • SourceForge.net: Área de download

    Este é o principal local de download do FolhaLivre. É uma área pública, disponibilizada e mantida pelo SourceForge.net, com mirror e estatísticas dos downloads realizados.

    Neste espaço estão disponibilizados os binários, a base de dados, manual e as fontes do FolhaLivre. Recomendamos que você sempre baixe os arquivos mais recentes.

    Se você desejar disponibilizar aqueles arquivos em sua página ou site, utilize os links para a área de download do Projeto FolhaLivre no SourceForge.net.

  • Yahoo!Grupos: Área de arquivos

    Esta é a área de arquivos do grupo de discusão folha_livre. É uma área restrita aos associados do grupo, disponibilizada e mantida pelo Yahoo!. Neste espaço estão disponibilizados arquivos de bibliotecas, documentos relacionados, especificações e textos diversos.

    Lembramos que para ter acesso essa área é obrigatório que você esteja logado no Yahoo!Grupos. Esclarecemos que essa obrigatoriedade não faz parte da política do Grupo folha_livre e sim do Yahoo!Grupos.

  • Repositório CVS: http://folha-livre.cvs.sourceforge.net/folha-livre/

    Se você é desenvolvedor ou deseja trabalhar com a versão de desenvolvimento do FolhaLivre é possível obter o código-fonte via CVS (Concurrent Versions System). Para mais detalhes siga para a página CVS na seção Obtendo arquivos do CVS.

Conhecendo os arquivos

A partir da versão 1.5, o conjunto de arquivos disponíveis para download foi modificado com o objetivo de facilitar o trabalho de obtenção e a instalação do sistema para novos usuários e desenvolvedores do FolhaLivre.

Os nomes desses arquivos possuem um sufixo no formato x.y, onde x.y é o número da versão estável. Os arquivos são:

  • flivre-install-x.y.zip (novo usuário)

    Este arquivo contém o binário (flivre.exe), a base de dados (flivre.fdb), os papéis de parede (.jpg), arquivo de configuração (flivre.ini), histórico (history.txt), a licença GNU-GPL (gnu-gpl.txt) e manual (.html). É ideal para os novos usuários e para aqueles desejam apenas utilizar ou testar o sistema;

  • flivre-upgrade-x.y.zip (atualização)

    Este arquivo contém o binário (flivre.exe), os papéis de parede (.jpg), o histórico (.txt), licença GNU-GPL (gnu-gpl.txt) e manual (.html). Os usuários que desejam atualizar seu sistema precisam apenas desse arquivo;

  • flivre-manual-x.y.zip (escritores)

    Este arquivo contém toda a infra-estrutura para produção do manual do FolhaLivre. É ideal para pessoas que desejam estudar e ajudar na confecção da documentação;

  • flivre-source-x.y.zip (desenvolvedores)

    Este arquivo contém o código-fonte completo do FolhaLivre, sem os arquivos de controle do CVS (pastas CVS). Os códigos-fontes das bibliotecas de terceiros não estão incluídos neste arquivo.

  • flivre-full-x.y.zip (desenvolvedores)

    Este arquivo contém todos os arquivos do FolhaLivre. Na verdade, este arquivo contém uma cópia da área de trabalho de desenvolvimento do FolhaLivre.

Agora que você já conhece os arquivos do FolhaLivre, chegou a hora de obtê-los !!!

Obtendo os arquivos (download)

Todos os arquivos acima mencionados estão disponíveis na área de download do SourceForge.net. Este é o principal local de download do FolhaLivre. É uma área pública, disponibilizada e mantida pelo SourceForge.net, com mirror e estatísticas dos downloads realizados.

A versão atual do FolhaLivre é a 1.5 (fevereiro/08). Portanto, os arquivos são:

  • flivre-install-1.5.zip
  • flivre-upgrade-1.5.zip
  • flivre-manual-1.5.zip
  • flivre-source-1.5.zip
  • flivre-full-1.5.zip

Depois de obter (baixar) os arquivos de seu interesse, leia antes o Manual de Instalação para preparar o seu ambiente para trabalhar com o FolhaLivre.
