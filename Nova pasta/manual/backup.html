Backup - Cópia de segurança

« Volta para índice

A cópia de segurança é um recurso dispónivel para que seja guardada uma cópia da base de dados a fim de que a mesma possa ser recuperada quando os dados originais ficarem danificados.

A cópia de segurança pode ser realizada mesmo que outros usuários estejam utilizando o sistema.

Siga os sequintes passos para a realização da cópia de segurança:

  1. Acesse a opção "Cópia de Segurança" do Menu "Ferramentas".

  2. Na tela sequinte mostrada é sugerido o nome do arquivo que conterá a cópia de segurança. Este nome é formado pelo nome da base de dados mais a data atual no formato dd-mm-yyyy com a extensão ".gbk".

    O local deste arquivo sempre é relativo ao SERVIDOR e não ao computador de onde está sendo realizado a cópia.

  3. Pressione o botão "Iniciar a cópia" e aguarde a conclusão da mesma.

  4. Quando a cópia estiver concluída será apresentada uma mensagem de aviso, pressione "OK" e a tela de cópia será fechada.

  5. O arquivo .gbk é gravado no mesmo diretório (pasta) da base de dados. Portanto, para uma maior segurança mova a cópia de segurança para um local mais apropriado.
