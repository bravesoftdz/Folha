==========================
Histórico das modificações
==========================

IMPORTANTE

   Após atualizar a versão binária, você dever ir para a opção
   >Ferramentas>Atualizar Estrutura da Base de Dados e confirmar a atualização.

Versão 1.5 - 09/02/2008
=======================

- Lista de incidências para os Totalizadores;
- Adicionado "Natureza de Rendimento" para Funcionário;
- A fonte "Arial" tornar-se a fonte padrão;
- Request ID 1815037 - Adicionado o Comprovante de Rendimentos;
- Request ID 1787538 - Novos campos para funcionários;
- Request ID 1730916 - Verificar código do novo funcionário;
- Request ID 1717933 - Cadastro de Recursos;
- Adicionado a guia "Grupos" no cadastro de eventos;
- Adicionado "Previsão de Funcionário" no cadastro de folhas;
- Reformulação da guia "Sequência de Cálculo" no cadastro de folhas;
- Adicionado o controle de **transação** (start,commit,rollback) no calculador;
- Início da utilização do FortesReport (gerador de relatório);
- Novo formato do arquivo de configuração flivre.ini;
- Adicionado o suporte para drivers DBExpress para desenvolvedores (versão beta).

Versão 1.4 - 20/04/2007
=======================

- Request ID 1686344 - Eventos Programados;
- Adicionado o importador para eventos informados;
- Calculador: Adicionado a função SN - Salario Normal (desconsidera o valor informado)
- Calculador: Adicionada a função DPU - Número de dias úteis no período da folha;
- Calculador: Adicionada a função DPF - Número de dias não-úteis no período da folha;
- Calculador: Adicionada a função GN - Gratificação Natalina;
- Calculador: Adicionada a função MM - Meses para Média;
- Calculador: Adicionada a função MAA - Meses de Admissão no Ano; 
- Bug fix: 1672060 - Salário Mensal e Salário;
- Bug Fix: 1666112 - Testar Valores Minimo e Maximo;
- Adicionado o campo "TIPO_SALARIO" no cadastro de eventos, para suporte da função GN();
- Adicionado o filtro por funcionário na impressão de contra-cheques;
- Melhorias no cadastro de folhas de pagamento;
- Melhorias no cadastro de eventos;
- Melhorias no cadastro de funcionários.
